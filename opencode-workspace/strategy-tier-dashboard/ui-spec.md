# 次高战略看板 页面视觉规格

## 1. 输入来源

- **需求材料**：用户口述《次高战略看板需求说明》
- **requirement.md**：`./requirement.md`
- **reqdoc.md**：`./reqdoc.md`
- **目标项目**：华润雪花营销驾驶舱 H5（`marketing-cockpit-dashboard-h5`）
- **同类页面**：营销驾驶舱「销量快报」数据看板（最直接的结构 / 视觉基线）
- **项目本地规则 / skills**：`snow-design-mobile/references/project-marketing-cockpit.md`

---

## 2. 页面骨架结论

- **阅读方式**：分屏看板型（顶部一级页签切战略群，Banner 内时间页签切口径，组织分段器切层级，表格看排名）
- **采用理由**：
  - 需求明确为 4 个一级页签（整体 / 1 / 3 / N），每个页签内汇总 + 组织表结构一致
  - 3 新三品页签内需 品种 × 规格 二维展开，是通用表格的变体
  - 与营销驾驶舱「销量快报」现有看板模式一致：顶部 Tab → Banner → 汇总卡 → 组织分段器 → 表格
- **按项目基线推断**：
  - Banner 内时间切换采用胶囊标签卡（Banner 内维度 Tab）
  - 组织切换采用分段器（与销量快报一致）
  - 表格采用固定首列 + 横向滚动 + 进度条
  - 3 页签表格采用两级分组表头（品种为大表头，规格为子表头）
- **项目覆盖**：本页对齐 `project-marketing-cockpit.md`，沿用紧凑数据布局；Banner 部分按用户指定采用 skill 看板 Hero 样式（COS 背景图层 + 正文内容流），高度 200px

## 3. 页面整体结构

```
顶部导航栏（次高战略看板 + 返回）
    ↓
页面级一级 Tab（整体 | 1(SuperX) | 3(新三品) | N(其他)）
    ↓
Banner 区域（标题 + 时间切换页签 当日|当月|年累计 + 时间文字）
    ↓
汇总卡片（当期 | 同期 | 同比增减 | 同比增幅）
    ↓
组织切换分段器（分事业部 | 分营销中心）
    ↓
数据表格（整体 / 1 / N：通用单层表；3：品种×规格二维分组表）
    ↓
（时间选择器抽屉，按需弹出）
```

## 4. 各页签详细结构

### 4.1 整体 Tab

- **Banner 标题**：次高战略看板 · 整体
- **汇总卡片**：当期、同期、同比增减、同比增幅（千升）
- **组织分段器**：分事业部 | 分营销中心
- **数据表格**：组织排名 + 当期 / 同期 / 同比增减 / 同比增幅 / 增量贡献（进度条）
- **说明**：整体 = 1 + 3 + N 合计；数据由后端直接返回合计

### 4.2 1(SuperX) Tab

- **Banner 标题**：次高战略看板 · 1(SuperX)
- **汇总卡片**：同整体结构，范围为 SuperX
- **组织分段器**：分事业部 | 分营销中心
- **数据表格**：通用单层表，同整体列结构

### 4.3 3(新三品) Tab

- **Banner 标题**：次高战略看板 · 3(新三品)
- **汇总卡片**：同整体结构，范围为新三品合计
- **组织分段器**：分事业部 | 分营销中心
- **二维数据表格**：
  - 固定首列：组织名 + 排名
  - 列分组：品种（雪花金冠 / 德式小麦 / 勇闯超干）为大表头，每组下 500 瓶 / 460 听 为子表头
  - 单元格：当前所选指标的销量（千升）
  - 表头指标切换：当期 | 同期 | 同比增减 | 同比增幅，6 规格列同步切换
  - 排名：按该组织新三品合计当期降序

### 4.4 N(其他) Tab

- **Banner 标题**：次高战略看板 · N(其他)
- **汇总卡片**：同整体结构，范围为其他产品
- **组织分段器**：分事业部 | 分营销中心
- **数据表格**：通用单层表，同整体列结构

## 5. 模块视觉规格

### 5.1 顶部导航栏

- 高度：44px
- 背景：White（#FFFFFF）
- 标题：次高战略看板，17/Bold，Text-5（#0B0D0F），居中
- 返回按钮：左侧箭头，24x24

### 5.2 页面级一级 Tab

- 容器：375 x 44，横向排列
- Tab 数量：4 个（整体 | 1(SuperX) | 3(新三品) | N(其他)）
- 选中项：16/Medium，Brand-6（#0073E5）
- 未选中项：14/Regular，Text-4（#576675）
- 选中指示线：24 x 2，圆角 1px，Brand-6
- 交互：点击切换，保留时间口径与组织页签状态
- 文案规则：页签文字带括号说明（1(SuperX)、3(新三品)、N(其他)），整体不带括号

### 5.3 Banner 区域（Hero 图层 + 正文内容流，高 200px）

- 结构：背景图层 `<img>` + 局部弱遮罩 + 正文内容流（标题 / 时间页签 / 时间文字）
- 容器高度：200px，overflow:hidden
- 背景图层：独立 `<img>`，position:absolute，inset:0，width:100%，height:100%，object-fit:cover（用户指定 200px 固定高度，故用 cover 填充；偏离 skill 默认 height:auto）
- 背景图来源：COS 素材池 `https://ai-1259627966.cos.ap-shanghai.myqcloud.com/前端资源目录/`（当前用示例图 `image-2@3x.png` 占位，正式上线从素材池随机选 1 张）
- 局部弱遮罩：`linear-gradient(180deg, rgba(0,30,70,.30), rgba(0,30,70,.55))`，仅保证文字可读，不覆盖原图主色
- 正文内容流：position:relative，z-index:2，padding 22px 16px 16px，纵向两端分布（标题 / 时间页签 / 时间文字）
- 内容：
  - Banner 主标题：当前战略群名（如「次高战略看板 · 整体」），18/Medium，白色
  - 时间切换页签（Banner 内维度 Tab，胶囊标签卡）：
    - 三项：当日 | 当月 | 年累计
    - 选中态：白色填充背景，Brand-6 文字，14/Medium
    - 未选中态：透明背景 + 白色描边，白色文字 90%，14/Regular
    - 圆角：999px，内边距 5px 14px，间距 8px
    - 默认选中：当月
  - 时间文字：当前所选日期（如 2026-06-29），12/Regular，白色 85%，可点击打开选择器

### 5.4 汇总卡片

- 背景：White（#FFFFFF）
- 圆角：8px
- 外边距：12px 16px
- 内边距：16px
- 布局：横向四等分排列
- 指标名称：12/Regular，Text-3（#8A99A8）
- 指标数值：20/Medium，Text-5（#0B0D0F），DIN 字体
- 单位：千升，附在数值右下角小字
- 同比增减 / 增幅正负色：正 = Success-4（#2EB82E），负 = Danger-4（#F53D3D），持平 = Text-3

### 5.5 组织切换分段器

- 容器：圆角 4px，1px Brand-6 描边
- 高度：36px
- 外边距：0 16px 12px
- 选中项：Brand-6 背景，白色文字，14/Medium
- 未选中项：透明背景，Brand-6 文字，14/Regular
- 交互：切换层级，保留战略群与时间口径

### 5.6 通用数据表格（整体 / 1 / N）

- 背景：White（#FFFFFF）
- 圆角：8px
- 外边距：0 16px 16px
- 表头背景：Brand-1（#E6F2FF）
- 表头文字：11/Regular，Text-4（#576675）
- 数据文本：13/Regular，Text-5（#0B0D0F）
- 列：组织名 | 当期 | 同期 | 同比增减 | 同比增幅 | 增量贡献（进度条）
- 固定首列：组织名 + 排名徽章，宽 88-96px
- 横向滚动：overflow-x:auto，隐藏滚动条
- 进度条：高 6px，Fill-2（#F1F1F4）背景，Brand-6 填充
- 排名 Top3：蓝色圆形徽章（Brand-6 背景，白色数字）；4 名后普通序号
- 正负色：同比增减 / 增幅按正负着色
- 单位：千升，保留 1 位小数

### 5.7 3 新三品二维数据表格

- 背景 / 圆角 / 外边距 / 表头背景：同通用表格
- 表头两级分组：
  - 一级（品种）：雪花金冠 | 德式小麦 | 勇闯超干，跨两列居中，13/Medium，Text-5
  - 二级（规格）：500 瓶 | 460 听，11/Regular，Text-4
  - 左侧固定首列表头：组织
- 指标切换器：表格右上角或表头行，胶囊切换 当期 | 同期 | 同比增减 | 同比增幅，默认当期
- 单元格：13/Regular，Text-5，DIN 数字；无数据「-」
- 固定首列：组织名 + 排名，宽 88-96px
- 横向滚动：overflow-x:auto，隐藏滚动条；两级表头随横向滚动吸顶
- 排名：按该组织新三品合计当期降序，Top3 蓝色徽章
- 正负色：同比增减 / 增幅列按正负着色
- 列宽：每规格列约 64-72px，不压缩

### 5.8 时间选择器抽屉

- 从底部弹出的抽屉
- 遮罩：rgba(0,0,0,0.4)
- 弹窗背景：White，圆角 12px（仅顶部）
- 标题：选择日期
- 内容：年月日三级选择或预设快捷选项
- 底部：取消 | 确认

## 6. 弹窗交互

### 6.1 时间选择器

- 触发：点击 Banner 时间文字
- 弹出：底部抽屉
- 可选范围：不晚于今日
- 确认后：按所选日期 + 当前口径刷新

### 6.2 空状态 / 异常

- 数据为空：表格区空状态插画 + 文案「暂无数据」
- 接口异常：toast 提示「加载失败，请重试」

## 8. 项目本地规则应用

按 `project-marketing-cockpit.md` 项目本地经验，命中即应用：

- **scrollview-height-pitfall**：3 新三品二维表格 scroll-view + flex 不同时设 height:0 与 min-height:0；注意 iPhone 安全区。
- **swiper-sticky-cleanup**：若使用 swiper 切页签，外层切页时清理吸顶表头，吸顶状态由父级统一控制。
- **dealer-name-separate-row**：组织名过长时独占一行，数据行用空列占位对齐表头。
- **remove-percent-multiply**：同比增幅后端返回百分比时前端直接拼 %，不再 ×100。

## 9. 设计冻结规则

- demo 实现阶段不再向用户确认颜色、布局、组件选择或是否继续
- 缺失信息按同类页面（销量快报）、项目画像、Snow Mobile 规范推断
- **明确禁止的脑补**：
  - 不要把次高战略群脑补成通用 KPI 仪表盘（销售额、订单量、达成率、近 7 天趋势）
  - 不要使用随机渐变、大面积彩色卡片、emoji 图标、营销式 Hero
  - 不要用一堆卡片从上到下堆完所有信息，忽略页签 / 时间页签 / 组织分段器
  - 不要给 1 / N 页签套用 3 页签的二维结构
  - 不要用 PC px 思维，需有移动端 H5 / uni-app 尺寸节奏
