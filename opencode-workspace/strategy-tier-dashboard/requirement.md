# 次高战略看板 — 需求分析文档

> **版本**：V1.0
> **日期**：2026-06-29
> **编写人**：AI Assistant
> **需求来源**：用户口述《次高战略看板需求说明》

---

## 1. 需求类型与演进方式

- **需求类型**：传统需求（数据看板 / 报表展示）
- **演进方式**：`1 到 N`（营销驾驶舱 H5 已有「销量快报」等数据看板页面，本次为新增「次高战略看板」独立页面，聚焦次高战略产品群）

---

## 2. 项目背景

华润雪花啤酒营销驾驶舱 H5 是面向内部销售、业务部、营销中心及管理层的数据看板系统。现有「销量快报」已覆盖分档次 / 分品牌 / 分主力品种 / 其他品种的销量表现，其中「分档次」Tab 下有「次高生意表现」子维度，但只是一个汇总切片，无法对次高战略产品群内部结构做专门看板式跟踪。

本次新增「次高战略看板」页面，把次高战略产品群按战略分组拆为：

- **1 = SuperX**
- **3 = 新三品**（雪花金冠、德式小麦、勇闯超干）
- **N = 其他**（次高战略群内 1、3 以外的其他产品）

并支持「整体 = 1 + 3 + N」总览，按日 / 月 / 年时间口径，分事业部 / 营销中心查看当期、同期、同比增减、同比增幅，辅助管理层判断次高战略群各子群的生意健康度和增长贡献。

---

## 3. 目标用户

| 角色 | 使用场景 | 核心诉求 |
|------|----------|----------|
| 事业部管理层 | 查看次高战略群整体及 1/3/N 各子群销量表现 | 快速判断战略群结构是否健康，定位增长 / 下滑子群 |
| 营销中心人员 | 查看所辖区域组织在次高战略群的排名 | 横向对比各组织表现，定位优劣势组织 |
| 业务部人员 | 查看 3 新三品各品种分规格的销量明细 | 精细化运营，调整品种 / 规格策略 |

---

## 4. 角色与权限概览

- **数据范围**：按用户所属组织层级隔离（事业部 / 营销中心）
- **查看权限**：内部销售、业务部、营销中心、管理层等组织层级用户
- **权限边界**：本次需求不涉及编辑 / 操作权限，仅为数据查看
- **权限沿用**：沿用营销驾驶舱 H5 现有组织数据范围权限，不新增角色

---

## 5. 核心问题

1. 如何让用户在一屏内按「整体 / 1 / 3 / N」四个战略群切换查看次高销量表现？
2. 如何在 Banner 内提供「当日 / 当月 / 年累计」时间口径切换，并保留组织页签状态？
3. 如何分事业部 / 营销中心查看当期、同期、同比增减、同比增幅四列核心指标？
4. 如何在「3 新三品」页签内对雪花金冠 / 德式小麦 / 勇闯超干按「500 瓶 / 460 听」二维展开，且在移动端有限屏宽内可读？

## 7. 现有流程分析

营销驾驶舱 H5 已有「销量快报」数据看板，采用以下成熟模式，本次「次高战略看板」延续该模式以保持体验一致：

- **页面结构**：顶部导航栏 → 页面级 Tab → Banner 区域（标题 + 时间 + 筛选）→ 子维度横滑导航 → 汇总卡片 → 组织切换页签 → 数据表格
- **组织切换**：页面级分段器（分事业部 / 分营销中心）
- **表格展示**：固定首列（组织名 + 排名徽章）+ 横向滚动 + 增量贡献进度条
- **时间筛选**：年月日
- **视觉风格**：雪花蓝主色 `#0073E5`、灰蓝文字阶梯、浅灰背景 `#F7F7F8`、白底卡片、DIN 数字感

与销量快报的差异：

| 维度 | 销量快报 | 次高战略看板 |
|------|----------|--------------|
| 页签维度 | 档次 / 品牌 / 主力品种 / 其他品种 | 整体 / 1(SuperX) / 3(新三品) / N(其他) |
| 时间切换 | Banner 外时间筛选（年月日） | Banner 内时间页签（当日 / 当月 / 年累计） |
| 二维展开 | 无 | 3 新三品页签内 品种 × 规格 二维 |
| 聚焦范围 | 全品种销量 | 次高战略产品群 |

## 9. 关键流程

```
用户进入次高战略看板
    ↓
默认展示「整体」页签 + 「当月」时间口径
    ↓
Banner 显示「次高战略看板 · 整体」及时间切换页签（当日 / 当月 / 年累计）
    ↓
汇总卡片展示整体当期 / 同期 / 同比增减 / 同比增幅
    ↓
组织切换页签（分事业部 / 分营销中心）
    ↓
数据表格展示组织排名及当期 / 同期 / 同比增减 / 同比增幅 / 增量贡献
    ↓
用户可切换「1 / 3 / N」页签查看各战略子群
    ↓
切到「3 新三品」时，表格按 雪花金冠 / 德式小麦 / 勇闯超干 × 500 瓶 / 460 听 二维展开
    ↓
用户可在 Banner 切换 当日 / 年累计 时间口径
    ↓
用户可通过时间选择器切换具体年月日
```

## 10. 非功能要求

- **性能**：页面加载 < 2s，数据查询 < 1s
- **兼容性**：适配 iOS / Android 主流机型，最低支持 iOS 12 / Android 8
- **响应式**：以 375px 移动端为主要设计基准，同时检查 390px 和 430px
- **可访问性**：关键指标文字清晰可辨，3 新三品二维表格支持横向滚动且首列固定
- **可观测性**：接口耗时、异常需有日志，便于定位数据口径问题

---

## 11. 风险与待确认项

| 风险 / 问题 | 影响范围 | 处理策略 |
|-------------|----------|----------|
| 3 新三品 品种 × 规格 × 4 指标 列数多，移动端展示易溢出 | 3 页签 | 采用横向滚动 + 固定首列 + 列分组（品种为组标题），不强行压缩列宽 |
| 「当日」口径是否为今日单日销量 | 所有页签 | 待确认；暂按今日单日设计，战略看板日维度意义有限，默认进当月 |
| 「N(其他)」范围是次高群内 1/3 以外，还是全品种其他 | N 页签、整体 | 待确认；暂按次高战略群内 1/3 以外的其他产品 |
| 「整体」页签是否需要展示 1/3/N 结构占比 | 整体页签 | 待确认；暂按汇总 + 组织表，不做结构占比卡 |
| 排名按哪个指标排序 | 所有表格 | 暂按当期销量降序 |
| 增量贡献进度条口径 | 所有表格 | 待后端确认增量贡献计算口径（相对同期增量的占比） |
| 同比增幅 = 同比增减 / 同期，后端是否已算好 | 所有表格 | 沿用销量快报规则：后端返回百分比则前端直接拼 %，不再 ×100 |
