> **版本**：v1.1
> **日期**：2026-06-29

> **优先级**：一级页签与时间切换、组织页签、数据表格、3 新三品二维展开（P0）；时间选择器（P1）

## 1. 需求背景与目标

### 1.1 项目背景

华润雪花营销驾驶舱 H5 已有「销量快报」数据看板，覆盖分档次 / 分品牌 / 分主力品种 / 其他品种的销量表现。其中「分档次」Tab 下的「次高生意表现」仅是一个汇总切片，无法对次高战略产品群内部结构（1 SuperX、3 新三品、N 其他）做专门的看板式跟踪，管理层难以快速判断各战略子群的增长贡献和结构健康度。

本次新增「次高战略看板」独立页面，把次高战略产品群按战略分组拆为 1（SuperX）、3（新三品 = 雪花金冠 / 德式小麦 / 勇闯超干）、N（其他），并支持整体（1+3+N）总览，按日 / 月 / 年时间口径，分事业部 / 营销中心查看当期、同期、同比增减、同比增幅。

### 1.2 业务目标

- 把次高战略产品群的结构表现从「销量快报的一个子维度」升级为「独立看板」，提升管理层对战略群内部结构的可见度
- 通过 1 / 3 / N 分群对比，识别增长引擎和拖累项
- 通过 3 新三品 品种 × 规格 二维展开，支撑品种 / 规格级别的精细化运营

### 1.3 用户目标

- **事业部管理层**：一屏看清次高战略群整体及 1/3/N 各子群的当期、同期、同比，定位增长 / 下滑子群
- **营销中心人员**：查看所辖区域组织在次高战略群的排名，横向对比优劣势
- **业务部人员**：在 3 新三品页签查看各品种分 500 瓶 / 460 听 规格的销量明细

### 1.4 本轮要解决的问题

1. 一屏内按 整体 / 1 / 3 / N 四个战略群切换查看次高销量表现
2. Banner 内提供 当日 / 当月 / 年累计 时间口径切换
3. 分事业部 / 营销中心查看 当期 / 同期 / 同比增减 / 同比增幅
4. 3 新三品页签内 品种 × 规格 二维展开，且移动端可读

### 1.5 不在本轮解决的问题

- 数据导出、图表（折线 / 柱状）、下钻到客户 / 门店、实时刷新、多组织同时对比、销售额指标

### 2.2 核心流程

> **流程图文件**：暂以文字概述，必要时补 `【流程图】次高战略看板-核心流程.drawio`

流程概述：

- **主流程**：进入页面 → 默认「整体」页签 + 「当月」时间口径 → Banner 显示看板名与时间切换页签 → 汇总卡片展示四指标 → 组织页签（默认分事业部）→ 数据表格展示组织排名与四指标 + 增量贡献进度条
- **分支 1（切战略群）**：点 1 / 3 / N 页签 → 汇总卡 + 表格联动到对应子群；切到 3 时表格切换为 品种 × 规格 二维结构
- **分支 2（切时间口径）**：点 Banner 内 当日 / 年累计 → 汇总卡 + 表格按新口径刷新；组织页签状态保留
- **分支 3（切组织）**：点 分营销中心 → 表格行换成营销中心组织排名
- **异常路径**：数据为空 → 表格区显示空状态；接口超时 → toast 提示并可重试；无权限组织 → 不展示该组织行

## 3. 需求详情

### 3.1 页面框架与一级页签

#### 3.1.1 适用角色与权限

- **全部角色**：按所属组织层级查看数据，无编辑权限

#### 3.1.2 触发入口

**入口 1 — 营销驾驶舱 H5 首页 / 工作台「次高战略看板」入口（本期主入口）**：从驾驶舱首页或销量看板专区进入次高战略看板页面
**入口 2 — 销量快报「分档次 → 次高生意表现」下钻（辅助入口）**：从销量快报次高切片跳转，带战略群参数进入

#### 3.1.3 页面与视图

> **页面视觉规格**：参考 `ui-spec.md`
> **界面原型**：参考 `demo.html` → 默认进入「整体」页签查看。以下为关键界面说明：

**次高战略看板主页面**：

![次高战略看板-整体页签](images/次高战略看板-整体页签.png)

- 页面标题：次高战略看板
- 顶部一级页签：整体 | 1(SuperX) | 3(新三品) | N(其他)
- 默认选中：整体

#### 3.1.4 交互步骤

1. 进入页面，默认选中「整体」页签，「当月」时间口径
2. 点击页签切换 1 / 3 / N，汇总卡与表格联动刷新，组织页签与时间口径状态保留
3. 切到「3」时，表格区切换为品种 × 规格二维结构

#### 3.1.5 关键状态

| 状态 | 说明 / 展示方式 |
|------|----------------|
| 默认态 | 整体页签 + 当月口径 + 分事业部组织 |
| 切换战略群 | 选中页签高亮（雪花蓝下划线），内容区联动 |
| 数据加载中 | 表格区骨架屏 / loading |
| 数据为空 | 表格区空状态插画 + 文案 |

#### 3.1.6 业务规则

- **战略群定义**：整体 = 1 + 3 + N；1 = SuperX；3 = 新三品（雪花金冠 + 德式小麦 + 勇闯超干）；N = 次高群内 1、3 以外的其他产品（待确认）
- **页签状态保留**：切换战略群时，时间口径与组织页签选择保留，不重置

#### 3.1.7 数据输入输出

- **输入**：战略群标识（overall / g1 / g3 / gN）、时间口径（day / month / ytd）、组织层级（bu / mc）、时间日期
- **输出**：该战略群在指定口径下的汇总指标 + 组织排名明细

#### 3.1.8 异常或边界情况

- **某战略群在当日口径下数据为空**：表格区空状态，提示「当日暂无数据，可切换当月 / 年累计查看」
- **用户无某组织层级权限**：该组织行不展示，不报错

#### 3.1.9 开发必看

> 页签切换不要重置时间口径和组织页签状态，三个维度的状态需独立管理。整体页签的数据 = 1 + 3 + N 的合计，后端若分别返回需前端汇总时注意口径一致；优先让后端直接返回整体合计避免前端汇总误差。

#### 3.1.10 重点注意

> 「1 / 3 / N」是战略群代号，不是序号；UI 上需带说明文案（1=SuperX、3=新三品、N=其他），避免用户误读为排名。

#### 3.1.11 风险提醒

> 整体页签若由前端把 1/3/N 三份数据相加，存在口径不一致风险（如 N 的范围与 1/3 重叠）。需后端明确整体口径并直接返回合计。

### 3.2 Banner 与时间切换页签

#### 3.2.1 适用角色与权限

- **全部角色**：可查看与切换

#### 3.2.2 触发入口

**入口 1 — Banner 内时间页签（本期主入口）**：Banner 区域内的 当日 / 当月 / 年累计 胶囊页签
**入口 2 — 时间选择器（辅助入口）**：Banner 时间文字点击，弹出年月日选择器

#### 3.2.3 页面与视图

**Banner 区域**：

![次高战略看板-Banner与时间页签](images/次高战略看板-Banner与时间页签.png)

- Banner 主标题：当前战略群名（如「次高战略看板 · 整体」）
- 时间切换页签：当日 | 当月 | 年累计（胶囊标签卡，置于 Banner 内）
- 时间文字：当前所选日期（如 2026-06-29），可点击打开选择器
- 默认选中：当月

#### 3.2.4 交互步骤

1. 点击 当日 / 当月 / 年累计 切换时间口径，汇总卡与表格立即联动
2. 点击时间文字打开年月日选择器，选择后按所选日期刷新
3. 切换战略群时，时间口径选择保留

#### 3.2.5 关键状态

| 状态 | 说明 / 展示方式 |
|------|----------------|
| 当日 | 统计今日单日销量（待确认口径） |
| 当月 | 统计当月 1 日至所选日期的月累计 |
| 年累计 | 统计当年 1 月 1 日至所选日期的年累计 |

#### 3.2.6 业务规则

- **时间口径定义**：当日 = 今日单日；当月 = 当月月初至选定日累计；年累计 = 当年年初至选定日累计
- **默认口径**：当月（战略看板日维度波动大、意义有限，默认月维度）
- **同比口径**：当期 vs 去年同期；当月同比 = 今年当月 vs 去年同月；年累计同比 = 今年 YTD vs 去年同期 YTD

#### 3.2.7 数据输入输出

- **输入**：时间口径（day / month / ytd）、基准日期
- **输出**：对应口径下的当期、同期、同比增减、同比增幅

#### 3.2.8 异常或边界情况

- **当日数据未出**：当日页签数据为空，提示并建议切当月
- **跨年时间**：年累计跨自然年时按当年 1 月 1 日起算

#### 3.2.9 开发必看

> 同比的「同期」随时间口径变化：当日 → 去年同日；当月 → 去年同月同期累计；年累计 → 去年同期 YTD。后端接口需按口径返回正确的同期值，不要固定为去年同月。

#### 3.2.10 重点注意

> 时间页签放在 Banner 内（胶囊标签卡），不是页面顶部 Tab；不要和时间选择器混为一个控件。

#### 3.2.11 风险提醒

> 「当日」对战略看板价值有限，若后端不支持单日同期，可降级为隐藏当日页签，但需产品确认。

### 3.3 汇总卡片

#### 3.3.1 适用角色与权限

- **全部角色**

#### 3.3.2 触发入口

**入口 1 — 页签内容区顶部（本期主入口）**：Banner 下方、组织页签上方的汇总卡片

#### 3.3.3 页面与视图

**汇总卡片**：

![次高战略看板-汇总卡片](images/次高战略看板-汇总卡片.png)

- 指标：当期 | 同期 | 同比增减 | 同比增幅
- 单位：千升（kL）
- 同比增减 / 增幅：正绿负红

#### 3.3.4 交互步骤

1. 切换战略群 / 时间口径 / 组织页签时，汇总卡片随之联动刷新

#### 3.3.5 关键状态

| 状态 | 说明 / 展示方式 |
|------|----------------|
| 正增长 | 同比增减 / 增幅显示绿色 |
| 负增长 | 同比增减 / 增幅显示红色 |
| 持平 | 灰色，显示 0 / 0.0% |

#### 3.3.6 业务规则

- **汇总范围**：当前战略群在当前组织层级权限下的全部组织合计
- **同比增幅**：= 同比增减 / 同期；后端若返回百分比数值，前端直接拼 `%`，不再 ×100

#### 3.3.7 数据输入输出

- **输入**：战略群、时间口径、组织层级
- **输出**：四指标汇总数值

#### 3.3.8 异常或边界情况

- **同期为 0**：同比增幅无法计算，显示「-」
- **数据为空**：四指标显示「--」

#### 3.3.9 开发必看

> 同比增幅百分比口径沿用销量快报规则：后端返回百分比则前端直接拼 %，不要再 ×100。

#### 3.3.10 重点注意

> 汇总卡是当前战略群的合计，不是某个组织的；切组织页签时汇总卡仍是该层级的全量合计，不要误做成单组织数值。

#### 3.3.11 风险提醒

> 切组织页签时若误把汇总卡联动成单组织，会导致与表格合计不一致。

#### 3.4.11 风险提醒

### 3.5 数据表格（整体 / 1 / N 通用结构）

#### 3.5.1 适用角色与权限

- **全部角色**

#### 3.5.2 触发入口

**入口 1 — 页签内容区表格（本期主入口）**：组织页签下方的数据表格

#### 3.5.3 页面与视图

**数据表格**：

![次高战略看板-数据表格](images/次高战略看板-数据表格.png)

- 固定首列：组织名 + 排名徽章
- 数据列：当期 | 同期 | 同比增减 | 同比增幅 | 增量贡献（进度条）
- 横向滚动，隐藏滚动条

#### 3.5.4 交互步骤

1. 表格按当期销量降序排名，Top3 蓝色徽章
2. 横向滑动查看 同比 / 增量贡献 列
3. 增量贡献以进度条展示占比

#### 3.5.5 关键状态

| 状态 | 说明 / 展示方式 |
|------|----------------|
| Top1-3 | 蓝色圆形排名徽章 |
| 4 名及以后 | 普通数字序号 |
| 正增长 | 同比增减 / 增幅绿色 |
| 负增长 | 同比增减 / 增幅红色 |

#### 3.5.6 业务规则

- **排名**：按当期销量降序
- **增量贡献**：该组织同比增量占整体同比增量的比例（待后端确认口径）
- **单位**：千升，保留 1 位小数

#### 3.5.7 数据输入输出

- **输入**：战略群、时间口径、组织层级
- **输出**：组织排名列表 + 各组织四指标 + 增量贡献

#### 3.5.8 异常或边界情况

- **组织名过长**：独占一行或省略，数据行对齐表头（沿用 dealer-name-separate-row 规则）
- **同比基数为 0**：增幅显示「-」

#### 3.5.9 开发必看

> 表格固定首列 + 横向滚动，列数超过屏宽时保留滚动不压缩列宽；固定首列与滚动区行高必须一致。沿用销量快报表格组件，不要新造。

#### 3.5.10 重点注意

> 增量贡献进度条的口径需后端确认，不要前端自行用当期 / 同期估算。

#### 3.5.11 风险提醒

> 整体页签的增量贡献分母为整体同比增量；1/3/N 页签的分母为该子群同比增量，跨页签的进度条不可直接比较。

### 3.6 3 新三品二维展开表格

#### 3.6.1 适用角色与权限

- **业务部、管理层**：重点使用

#### 3.6.2 触发入口

**入口 1 — 「3 新三品」页签内表格（本期主入口）**：切到 3 页签时表格自动切换为二维结构

#### 3.6.3 页面与视图

**3 新三品二维表格**：

![次高战略看板-3新三品二维表](images/次高战略看板-3新三品二维表.png)

- 固定首列：组织名 + 排名
- 列分组：按品种分组（雪花金冠 / 德式小麦 / 勇闯超干），每组下分 500 瓶 / 460 听 两个规格子列
- 每个规格子列下展示当期销量（千升）；可通过表头切换 / 横滑查看 同期 / 同比增减 / 同比增幅
- 横向滚动，列分组表头吸顶

#### 3.6.4 交互步骤

1. 切到 3 页签，表格呈现 品种（3）× 规格（2）= 6 个数据列的二维结构
2. 横向滑动查看各品种规格的当期销量
3. 表头指标切换（当期 / 同期 / 同比增减 / 同比增幅）时，6 列数据同步切换指标
4. 排名按「该组织新三品合计当期」降序

#### 3.6.5 关键状态

| 状态 | 说明 / 展示方式 |
|------|----------------|
| 默认指标 | 当期 |
| 切指标 | 表头点击切换 当期 / 同期 / 同比 / 增幅，6 列同步 |
| 某品种规格无数据 | 单元格显示「-」 |

#### 3.6.6 业务规则

- **二维定义**：行 = 组织；列 = 品种 × 规格；单元 = 销量（千升）
- **品种顺序**：雪花金冠 → 德式小麦 → 勇闯超干
- **规格顺序**：500 瓶 → 460 听
- **排名**：按该组织新三品合计当期降序
- **指标切换**：表头切换指标时，6 个规格列同步切换，不单独切换

#### 3.6.7 数据输入输出

- **输入**：时间口径、组织层级、品种、规格
- **输出**：组织 × 品种 × 规格 的销量矩阵 + 各组织新三品合计

#### 3.6.8 异常或边界情况

- **某组织无某规格销量**：单元格「-」
- **列数多导致横向溢出**：保留横向滚动，不压缩列宽

#### 3.6.9 开发必看

> 3 页签表格列数 = 1（首列）+ 3 品种 × 2 规格 = 7 列起步，单指标下已较宽；切换指标时是同 6 列换数值，不是新增 6 列。表头采用两级分组（品种为大表头，规格为子表头），分组表头随横向滚动吸顶。沿用 scrollview-height-pitfall：scroll-view + flex 不要同时设 height:0 和 min-height:0。

#### 3.6.10 重点注意

> 二维展开仅 3 页签独有；1 / N / 整体 页签仍是通用单层列表格。不要把二维结构套到其他页签。

#### 3.6.11 风险提醒

> 移动端 7+ 列易溢出，需确保固定首列与滚动区行高一致、滚动顺畅；表头分组吸顶在 uni-app swiper 切页时需清理吸顶状态（swiper-sticky-cleanup）。

#### 3.7.1 适用角色与权限

- **全部角色**

#### 3.7.2 触发入口

#### 3.7.3 页面与视图

**时间选择器**：

![次高战略看板-时间选择器](images/次高战略看板-时间选择器.png)

- 年月日三级选择或预设快捷选项
- 底部：取消 | 确认

#### 3.7.4 交互步骤

1. 点击 Banner 时间文字，弹出底部选择器
2. 选择年月日，确认后按新日期刷新汇总卡与表格

#### 3.7.5 关键状态

| 状态 | 说明 / 展示方式 |
|------|----------------|
| 当日口径 | 选择某日，统计该日单日 |
| 当月口径 | 选择某日，统计该月月初至该日 |
| 年累计口径 | 选择某日，统计该年年初至该日 |

#### 3.7.6 业务规则

- **可选范围**：不晚于今日；最早按业务数据上线时间

- **输入**：所选年月日
- **输出**：刷新后的数据

#### 3.7.8 异常或边界情况

- **选择未来日期**：不可选或置灰

#### 3.7.9 开发必看

> 时间选择器与时间口径页签是两个控件：口径决定统计方式（日 / 月 / 年累计），选择器决定基准日期。不要合并。

#### 3.7.10 重点注意

> 沿用销量快报的时间选择器交互，降低用户学习成本。

#### 3.7.11 风险提醒

> 当日口径下选择历史日期，需后端支持任意历史日的单日与去年同日数据。

## 6. 风险与处理策略

| 风险 | 影响 | 处理策略 |
|------|------|----------|
| **需求理解风险**：整体口径是否后端直接返回合计 | 整体页签数据准确性 | 待确认；优先后端返回合计，避免前端汇总 1/3/N 误差 |
| **交互或流程风险**：三维状态未独立保留 | 切换时状态丢失 | 前端三维状态独立管理，切换不重置 |
| **技术依赖风险**：3 新三品矩阵列数多，移动端溢出 | 3 页签可读性 | 横向滚动 + 固定首列 + 两级分组表头，不压缩列宽 |
| **数据一致性风险**：同比增幅口径 ×100 | 指标显示错误 | 沿用销量快报：后端返回百分比则前端直接拼 % |
| **排期风险**：3 新三品二维表格复杂度高 | 排期 | 二维表格独立排期，可与通用表格并行 |
| **数据口径风险**：「当日」与「N 其他」范围未定 | 当日页签、N 页签 | 待确认；暂按今日单日、次高群内 1/3 以外设计 |
