## 1. 需求类型与演进方式

- **需求类型**：传统需求（数据看板 / 报表展示，无 AI 能力）

---

## 2. 项目背景

华润雪花啤酒营销驾驶舱 H5 是面向内部销售、业务部、营销中心及管理层的数据看板系统。现有看板已覆盖分档次、分品牌、分主力品种等常规啤酒销量维度，但**创新业务**（代工类、特色产品、桶啤）作为独立业务板块尚无专门看板。

管理层目前查看创新业务销量需通过线下报表或口头汇报，存在以下问题：
- 创新业务包含代工类、特色产品、桶啤三大板块，各板块下还有细分品类，线下报表难以一屏看清整体结构
- 缺少同期对比和同比增幅，无法快速判断各板块增长 / 下滑态势
- 无法按组织层级横向对比各区域 / 营销中心 / 工厂的创新业务表现

本次需求为在营销驾驶舱 H5 中新增"创新业务看板"模块，让管理层一屏看清创新业务大盘结构与组织表现。

## 3. 目标用户

| 角色 | 使用场景 | 核心诉求 |
|------|----------|----------|
| 总部管理层 | 每日 / 每月查看全国创新业务大盘 | 快速掌握代工类 / 特色产品 / 桶啤三大板块销量结构及同比变化 |
| 区域管理层 | 查看所辖区域创新业务排名 | 横向对比各区域创新业务表现，定位增长 / 下滑区域 |
| 营销中心负责人 | 查看所辖营销中心及工厂创新业务 | 了解下属各组织创新业务完成情况 |
| 业务部人员 | 查看具体品类创新业务销量 | 精细化跟踪代加工 / 包销 / 定制等细分品类表现 |

## 4. 角色与权限概览

### 4.1 组织层级

| 层级 | 说明 | 数据范围 |
|------|------|----------|
| 总部 | 全国视角 | 可看全国所有组织创新业务数据 |
| 区域 | 大区视角 | 仅看本区域及下属组织数据 |
| 营销中心 | 营销中心视角 | 仅看本营销中心及下属工厂数据 |
| 工厂 | 工厂视角 | 仅看本工厂数据 |

### 4.2 权限规则

- **数据查看权限**：按用户所属组织层级隔离，高层级可看低层级数据，低层级不可看高层级数据
- **组织排名可见范围**：用户只能看到自己权限范围内的组织排名
  - 总部用户：可看全国各区域排名
  - 区域用户：可看本区域各营销中心排名
  - 营销中心用户：可看本营销中心各工厂排名
- **操作权限**：本次需求仅为数据查看，不涉及编辑 / 导出 / 配置操作
- **权限沿用**：沿用现有营销驾驶舱 H5 的组织权限体系，不新增角色

### 4.3 待确认权限点

| 待确认项 | 影响范围 | 当前处理 |
|----------|----------|----------|
| 组织层级切换是页面页签还是顶部筛选器 | 组织排名表格结构 | 暂参考销量快报用页签切换，列待确认 |
| 工厂层级用户看到的组织排名是自身单条还是同级工厂列表 | 工厂用户视图 | 暂按"自身单条 + 上级汇总"设计，列待确认 |

## 5. 核心问题

1. 如何让管理层在一个页面内快速切换"整体 / 代工类 / 特色产品 / 桶啤"四大视角查看创新业务销量？
2. 如何直观展示各视角下当期、同期、同比增减、同比增幅四项核心指标？
3. 如何展示三大板块下的细分品类（代加工 / 包销 / 定制等）销量表现？
4. 如何支持组织层级切换（总部 / 区域 / 营销中心 / 工厂），实现横向排名对比？
5. 如何支持日 / 月 / 年三种时间口径切换，适配不同决策频率？

## 7. 现有流程分析（1 到 N）

### 7.1 营销驾驶舱 H5 现有看板模式

基于目标项目基线（`project-marketing-cockpit.md`）和同类页面（销量快报）代码取证，现有看板采用以下固定模式：

| 维度 | 现有模式 |
|------|----------|
| 页面结构 | 顶部导航栏 → 页面级 Tab → Banner 区域 → 子维度横滑 → 内容区 → 组织页签 → 数据表格 |
| 组织切换 | 页面级页签（如"分事业部 / 分营销中心"），切换后表格行数变化 |
| 表格展示 | 固定首列（组织名 + 排名徽章）+ 横向滚动 + 进度条（增量贡献） |
| 视觉风格 | 雪花蓝 `#0073E5`、灰蓝文字阶梯、浅灰背景 `#F7F7F8`、DIN 数字感 |
| 交互 | Tab 切换、横滑子维度、筛选弹窗（底部抽屉）、时间选择 |
| 数据口径 | 当期 / 同期 / 同比增减 / 同比增幅 / 增量贡献（进度条） |

### 7.2 销量快报（同类页面）结构参考

销量快报是营销驾驶舱中最接近的同类页面，其结构为：

```
顶部导航栏（销量快报 + 返回）
    ↓
页面级 Tab（分档次 | 分品牌 | 分主力品种 | 其他品种）
    ↓
Banner 区域（标题 + 时间 + 筛选按钮）
    ↓
子维度横滑导航
    ↓
汇总卡片（当期 / 同期 / 同比增幅）
    ↓
组织切换页签（分事业部 | 分营销中心）
    ↓
数据表格（组织排名 + 当期 / 同期 / 同比增减 / 同比增幅 / 增量贡献）
```

### 7.3 本次需求与现有模式的差异

| 对比项 | 销量快报（现有） | 创新业务看板（本次） |
|--------|------------------|----------------------|
| Tab 含义 | 按档次 / 品牌 / 品种维度切换 | 按创新业务板块切换（整体 / 代工类 / 特色产品 / 桶啤） |
| 子维度 | 横滑标签切换单品 | 子类分行列表（一个表格内） |
| 组织层级 | 分事业部 / 分营销中心 | 总部 / 区域 / 营销中心 / 工厂 |
| 时间口径 | 年 / 月 / 日 | 日 / 月 / 年 |
| 图表 | 无 | 无（保持一致） |
| 导航方式 | 页面级 Tab | Banner 内导航按钮 |

### 7.4 本次需求改动的环节

- **新增模块**：在营销驾驶舱中新增"创新业务看板"入口和页面
- **复用框架**：顶部导航栏、Banner 区域、组织页签、数据表格结构沿用现有模式
- **新增交互**：Banner 内导航按钮（区别于页面级 Tab）、四层分类层级（整体 → 三大板块 → 子类 → 组织）
- **组织层级扩展**：从"事业部 / 营销中心"两级扩展为"总部 / 区域 / 营销中心 / 工厂"四级

## 9. 关键流程

### 9.1 主流程

```
用户进入创新业务看板页面
    ↓
默认展示「整体」视角 + 当前月
    ↓
Banner 区域显示「创新业务看板」标题 + 时间 + 四大导航按钮
    ↓
三大分类汇总表格（代工类 / 特色产品 / 桶啤 × 当期 / 同期 / 同比增减 / 同比增幅）
    ↓
组织层级页签切换（总部 / 区域 / 营销中心 / 工厂）
    ↓
组织排名表格（组织名 + 当期 / 同期 / 同比增减 / 同比增幅 + 进度条）
    ↓
用户可切换 Banner 导航按钮查看其他视角
    ↓
用户可切换组织层级页签查看不同层级排名
    ↓
用户可切换日 / 月 / 年时间口径
```

### 9.2 视角切换流程

```
选中「整体」
    → 汇总表格：代工类 / 特色产品 / 桶啤（3 行）
    → 组织排名：各组织创新业务总量排名

选中「代工类」
    → 子类表格：代加工 / 包销 / 定制（3 行）
    → 组织排名：各组织代工类销量排名

选中「特色产品」
    → 子类表格：小品种 / 贸易产品 / 醴（3 行）
    → 组织排名：各组织特色产品销量排名

选中「桶啤」
    → 子类表格：高档桶啤 / 普通桶啤（2 行）
    → 组织排名：各组织桶啤销量排名
```

### 9.3 时间口径切换

| 口径 | 当期含义 | 同期含义 |
|------|----------|----------|
| 日 | 当日累计销量 | 去年同日销量 |
| 月 | 当月累计销量 | 去年同月销量 |
| 年 | 当年累计销量 | 去年同期累计销量 |

## 10. 非功能要求

| 维度 | 要求 |
|------|------|
| 性能 | 页面加载 < 2s，数据查询接口 < 1s，表格渲染 < 500ms |
| 兼容性 | 适配 iOS / Android 主流机型，最低支持 iOS 12 / Android 8 |
| 响应式 | 以 375px 移动端为主要设计基准，同时检查 390px 和 430px |
| 数据单位 | 销量统一使用千升（kL），保留 1 位小数 |
| 可访问性 | 关键指标文字清晰可辨，表格支持横向滚动，正负色区分明显 |
| 稳定性 | 接口超时 10s 后展示兜底提示，数据为空时展示空状态 |
| 可观测性 | 页面访问埋点、视角切换埋点、组织切换埋点 |

## 11. 风险与待确认项

| 风险 / 问题 | 影响范围 | 处理策略 |
|-------------|----------|----------|
| 组织层级切换是页签还是下拉筛选器 | 组织排名表格结构、交互设计 | 暂参考销量快报用页签（分段器），列待确认 |
| 工厂层级用户看到的组织排名内容 | 工厂用户视图 | 暂按"自身单条 + 上级汇总"设计，列待确认 |
| "当期"在日口径下是否为当日实时累计 | 数据口径、刷新频率 | 暂按 T+1 设计，列待确认 |
| 增量贡献进度条的计算口径 | 组织排名表格进度条 | 暂按"当期销量 / 当期最大组织销量"计算，需后端确认 |
| 创新业务数据是否已在现有数据中台落库 | 后端接口设计、数据来源 | 需确认数据来源系统，暂按营销驾驶舱统一数据中台设计 |
| 同比增幅为无穷大时（同期为 0）的展示 | 数据展示、异常处理 | 展示"—"或"NEW"，不作除法计算 |
| Banner 导航按钮与页面级 Tab 的关系 | 页面结构、与销量快报一致性 | 暂按 Banner 内导航按钮设计（用户明确要求），区别于销量快报的页面级 Tab |
| 日 / 月 / 年切换是否影响组织排名表格列结构 | 表格列设计 | 暂按不影响（列固定为当期 / 同期 / 同比增减 / 同比增幅），列待确认 |
