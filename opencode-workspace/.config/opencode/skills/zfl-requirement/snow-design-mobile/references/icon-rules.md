# 图标规则

## 图标资产渠道

只允许使用两类图标资产：

1. Remix Icon：用于开源图标库选型，地址 `https://remixicon.com/`。
2. AI 生成：用于 Snow Mobile 首页、工作台、功能入口等需要面性彩色图标的场景。

不要使用其他开源图标库、随机网络图片或临时拼凑的装饰 SVG。

## 图标类型

| 类型 | 用法 | 推荐渠道 |
| --- | --- | --- |
| 导航图标 | 底部 Tab、顶部操作、工具入口 | Remix Icon |
| 功能入口图标 | 首页宫格、工作台入口、业务快捷入口 | AI 生成；低保真阶段可用 Remix Icon 占位 |
| 数据指标图标 | 看板 KPI、统计卡片、指标模块 | Remix Icon 或 AI 生成 |
| 状态图标 | 空状态、成功、失败、异常、无权限 | Remix Icon 或 AI 生成 |
| 业务插图图标 | 结果页、引导页、复杂空状态 | AI 生成 |

## 图标自动推断规则

当 PRD 或用户需求没有明确描述图标时，仍要根据页面结构判断哪些位置需要图标。图标只用于提升识别效率、表达状态或强化关键入口，不作为普通装饰。

生成页面前先做图标位盘点：

- 列出必须有图标的位置。
- 列出可以有图标但不是必须的位置。
- 列出不应加图标的位置。
- 为每个图标位确定资产渠道：Remix Icon、AI 生成或语义占位键名。
- 若使用语义占位键名，页面视觉中仍应使用 Remix Icon 占位，不能直接显示键名或文字代替图标。

默认需要考虑图标的位置：

- 首页、工作台、功能宫格、业务快捷入口。
- 底部 Tab、顶部工具、页面级操作入口。
- 看板 KPI、数据指标卡、统计卡片，例如销售额、订单数、达成率、库存、异常数。
- 排行榜前 3 名，例如销量、门店、区域、人员、商品排名。
- 空状态、成功、失败、异常、无权限、网络错误等反馈状态。
- 审批、配送、拜访、订单流转等流程节点。
- 搜索、筛选、扫码、新增、导出、刷新、更多等工具操作。
- 异常、预警、逾期、待处理等风险提醒。

默认不要加图标的位置：

- 普通字段标签。
- 列表行里的每一个元信息字段。
- 长说明文案、帮助文本、备注文本。
- 普通表单项，除非字段本身强依赖搜索、日期、定位、扫码等明确符号。
- 已经通过徽标、颜色、状态文案清楚表达的地方。

推断后的图标选型：

- 首页、工作台、业务入口使用 AI 生成的面性彩色功能图标。
- KPI 和数据指标卡使用“浅色圆角方底 + 无底色图标主体”。
- 排行榜、Top N、排名列表的第 1-3 名必须使用奖杯、皇冠、奖牌或排名徽章；第 4 名以后使用普通数字序号。
- 空状态、结果页和异常状态使用状态图标或 AI 生成业务插图。
- 搜索、筛选、扫码、新增、导出、刷新、更多等工具操作使用 Remix Icon。
- 流程节点优先使用小尺寸语义图标或节点圆点，不要做成复杂插画。

密度控制：

- 图标放在结构性位置，不要分散到每个小字段旁边。
- 每个 KPI 卡最多 1 个图标。
- 每条列表行最多 1 个主图标，普通业务列表通常可以不加。
- 排行榜只强化前 3 名，第 4 名以后保持弱化；前 3 名缺少排名图标或排名徽章视为不合格。
- 同一模块内图标风格、尺寸、底色和语义强度保持一致。
- 如果模块已经有徽标、状态色、趋势箭头或强状态文案，不再叠加装饰性图标。

## 禁止文字冒充图标

文字不能作为图标交付。

禁止：

- 用单个汉字代替功能入口图标，例如“订”“促”“配”“售”“账”“库”“券”“价”。
- 用单个汉字代替底部 Tab 图标，例如“首”“商”“单”“我”。
- 把文字放进圆形、圆角方形或彩色底色里，伪装成功能图标。
- 用业务缩写、首字母、数字或 emoji 代替图标。
- 在低保真页面中用文字占位后不说明替换方式。

允许：

- 低保真阶段使用 Remix Icon 作为视觉占位，并保留明确图标名。
- 输出 `icon: order-management` 这类语义占位键名作为交付说明。
- 在说明中标注“该位置后续替换为 AI 生成面性彩色功能图标”。

如果没有真实功能图标资产，页面视觉优先使用 Remix Icon 占位；交付说明中写清楚图标语义和后续替换为 AI 生成图标。

## Remix Icon 使用规则

- 优先使用 `-fill` 面性图标，只有导航默认态、工具弱入口或低强调场景才使用 `-line`。
- Remix Icon 适合系统动作、导航、状态和低保真功能占位。
- Remix Icon 默认是单色符号，不能直接当作首页面性彩色功能图标的最终资产。
- 使用 Remix Icon 做功能入口占位时，要保留明确图标名，例如 `remix:ri-shopping-bag-3-fill`、`remix:ri-store-2-fill`。
- 同一页面内 Remix Icon 的线宽、填充风格和尺寸必须统一。
- 不要混用其他图标库的命名、风格或图标。

## AI 生成功能图标规则

首页、工作台和业务功能入口优先使用面性彩色图标。

风格：

- 面性彩色、轻拟物或扁平插画。
- 主体图形清晰，最多 2-3 个业务元素。
- 不使用纯线框图标作为最终功能入口图标。
- 不在图标内部放文字，入口名称由页面文案承载。

颜色：

- 主色优先使用 Snow 品牌蓝或业务语义色。
- 辅色控制在 1-2 个，避免彩虹色。
- 同一页面的图标明度、饱和度和光影强度保持一致。
- 状态类图标必须遵守成功、警告、危险、中性语义。

底色：

- 首页功能入口允许使用浅色圆形底、圆角方形底或无底色图标。
- 同一组入口的底色形状必须统一。
- 底色应使用浅色、低强调，不抢入口标题和主体图形。
- 使用浅色圆角方底时，底色必须与图标主体颜色属于同一色系：蓝色图标配浅蓝底，绿色图标配浅绿底，橙色图标配浅橙底，红色图标配浅红底。
- 不要出现图标主体与底色语义错配，例如橙色图标配浅绿色底、红色图标配浅蓝色底。
- 有分类分组时，可用不同浅色底表达业务域，但不能让颜色成为唯一识别方式。

## 看板数据指标图标

看板、数据卡片、统计指标模块中的图标统一使用“浅色圆角方底 + 无底色图标主体”。

必备规则：

- 看板 KPI、数据指标卡、统计卡片默认必须配置 1 个指标图标。
- 指标图标不是装饰项，而是 KPI 卡片结构的一部分。
- PRD 未描述图标时，也要根据指标名称和业务语义自动选择图标。
- 缺少指标图标的 KPI 卡片视为不符合 Snow Mobile 看板规范。
- 只有趋势图、风险提醒、明细列表等非指标卡模块可以不配置 KPI 图标。

结构：

- 图标外层使用浅色圆角方形底。
- 图标主体不再自带底色、圆形底或额外装饰底。
- 图标主体居中，保持简洁，不做复杂插画。
- 同一看板内所有指标图标的底色容器尺寸、圆角、图标尺寸保持一致。

尺寸：

| 场景 | 底色容器 | 图标主体 | 圆角 |
| --- | ---: | ---: | --- |
| 常规指标卡 | 32 或 36 | 18 或 20 | Radius-8 |
| 强调指标卡 | 40 | 22 或 24 | Radius-8 |

颜色：

- 底色使用对应业务语义的浅色背景，优先从 `Brand-1`、`Success-1`、`Warning-1`、`Danger-1` 中选择。
- 图标主体使用对应业务语义的常规色，优先从 `主要颜色/primary`、`Success-4`、`Warning-4`、`Danger-4` 中选择。
- 底色和图标主体必须使用同一语义色系：`Brand-1` 对应 `主要颜色/primary`，`Success-1` 对应 `Success-4`，`Warning-1` 对应 `Warning-4`，`Danger-1` 对应 `Danger-4`。

使用规则：

- 销售额、订单数、达成率、库存、异常数等 KPI 图标使用此规则。
- 不使用首页功能入口那种更丰富的面性彩色插画。
- 不使用无底色裸图标直接放在数据卡片里。
- 不在指标图标内部放文字、数字或单位。
- 指标图标只辅助识别，不能比指标数字更抢眼。

尺寸：

| 场景 | 图标尺寸 | 底色容器 | 说明 |
| --- | ---: | ---: | --- |
| 首页宫格 | 40 或 48 | 48 或 56 | 常规功能入口 |
| 工作台快捷入口 | 40 | 48 | 信息密度较高 |
| 大型业务入口 | 56 | 64 | 少量重点入口 |
| 状态/结果页 | 80-120 | 按页面结构 | 可更具象 |

安全区：

- 图标主体与边界保留约 12%-16% 安全区。
- 关键业务符号不要贴边、压缩或被底色裁切。
- 图标在 1x 和 2x 下都要清晰。

## AI 生成 Prompt 模板

生成单个功能图标时使用：

```text
Snow Mobile 内部业务 APP 功能入口图标，面性彩色，轻拟物扁平风格，主体为「{业务对象/动作}」，包含不超过 2-3 个业务元素，Snow 品牌蓝为主色，辅色 1-2 个，浅色圆角底，移动端首页宫格图标，图标内不含文字，主体居中，留 12%-16% 安全区，干净统一，可在 48px 尺寸下识别。
```

批量生成同一页面图标时使用：

```text
为 Snow Mobile 内部业务 APP 生成一组功能入口图标，面性彩色，轻拟物扁平风格，统一视角、统一光影、统一圆角浅色底。业务入口包括：{入口列表}。每个图标主体清晰，不超过 2-3 个业务元素，图标内不含文字，Snow 品牌蓝为主色，辅色克制，适合 48px 首页宫格展示。
```

## 占位与交付

- 低保真页面可以使用 Remix Icon 占位，但必须标注需要替换为 Snow 面性彩色功能图标。
- 最终稿不能把 Remix Icon 单色符号直接当作首页功能图标。
- 如果图标资产尚未生成，输出占位键名：`icon: order-management`、`icon: store-visit`、`icon: sales-dashboard`。
- 同一页面的功能图标要成组生成或成组选择，避免单个图标风格孤立。
- 交付时说明每个功能入口的图标语义，不只说明视觉效果。
