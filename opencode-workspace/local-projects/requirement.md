# 销量快报 — 需求分析文档

> **版本**：V1.0
> **日期**：2026-06-24
> **编写人**：AI Assistant
> **需求来源**：王乾《销量快报需求说明》

---

## 1. 需求类型与演进方式

- **需求类型**：传统需求（数据看板/报表展示）
- **演进方式**：`1 到 N`（营销驾驶舱 H5 已有类似数据看板页面，本次为新增"销量快报"模块）

---

## 2. 项目背景

华润雪花啤酒营销驾驶舱 H5 是面向内部销售、业务部、营销中心及管理层的数据看板系统。本次需求为新增"销量快报"模块，支持按多维度（档次、品牌、主力品种、其他品种）快速查看销量生意表现，辅助业务决策。

---

## 3. 目标用户

| 角色 | 使用场景 | 核心诉求 |
|------|----------|----------|
| 事业部管理层 | 查看各档次/品牌/品种整体销量表现 | 快速掌握生意大盘，识别增长/下滑维度 |
| 营销中心人员 | 查看所辖区域组织排名 | 横向对比各组织表现，定位优劣势 |
| 业务部人员 | 查看具体品种销量占比和增幅 | 精细化运营，调整销售策略 |

---

## 4. 角色与权限概览

- **数据范围**：按用户所属组织层级隔离（事业部 / 营销中心）
- **查看权限**：内部销售、业务部、营销中心、管理层等组织层级用户
- **权限边界**：本次需求不涉及编辑/操作权限，仅为数据查看

---

## 5. 核心问题

1. 如何让用户在一个页面内快速切换不同维度（档次/品牌/品种）查看销量表现？
2. 如何直观展示各维度下的当期、同期、同比等核心指标？
3. 如何支持组织层级切换（事业部/营销中心），实现横向对比？
4. 如何在移动端有限屏幕内高效展示多列数据表格？

---

## 6. 目标与非目标

### 目标（本期必做）

- [x] 四个主 Tab 切换：分档次、分品牌、分主力品种、其他品种
- [x] 每个 Tab 内 Banner 横幅展示维度说明
- [x] 汇总卡片展示核心指标（当期、同期、同比增幅）
- [x] 组织切换页签（分事业部 / 分营销中心）
- [x] 数据表格展示组织排名及详细指标
- [x] 时间筛选（年月日）
- [x] 分档次/分主力品种/其他品种的品种筛选

### 非目标（本期不做）

- [ ] 数据导出功能
- [ ] 图表（折线图/柱状图等）展示
- [ ] 数据下钻到具体客户/门店
- [ ] 实时数据刷新/推送
- [ ] 多组织层级同时对比

## 7. 现有流程分析

营销驾驶舱 H5 已有类似数据看板页面，采用以下模式：

- **页面结构**：顶部 Tab 切换维度 → Banner 区域 → 汇总卡片 → 数据表格
- **组织切换**：页面级 Tab（分事业部 / 分营销中心）
- **表格展示**：横向滚动，固定首列（组织名称），支持进度条展示增量贡献
- **视觉风格**：雪花蓝主色、紧凑数据布局、DIN 数字感

本次"销量快报"应延续上述模式，保持与现有页面一致的用户体验。

| P0 | 汇总卡片 | 当期、同期、同比增幅核心指标 |
| P0 | 组织切换页签 | 分事业部 / 分营销中心 |
| P0 | 数据表格 | 组织排名 + 多指标 + 进度条 |
| P1 | 时间筛选 | 年月日切换 |
| P1 | 品种筛选 | 分档次/分主力品种/其他品种下的筛选 |

---

## 9. 关键流程

```
用户进入销量快报页面
    ↓
默认展示「分档次」Tab
    ↓
Banner 区域显示「分档次生意表现」及维度说明
    ↓
汇总卡片展示核心指标（当期/同期/同比增幅）
    ↓
组织切换页签（分事业部/分营销中心）
    ↓
数据表格展示组织排名及详细指标
    ↓
用户可切换顶部 Tab 查看其他维度
    ↓
用户可切换组织页签查看不同层级数据
    ↓
用户可通过筛选按钮筛选具体档次/品种
```

---

## 10. 非功能要求

- **性能**：页面加载 < 2s，数据查询 < 1s
- **兼容性**：适配 iOS/Android 主流机型，最低支持 iOS 12 / Android 8
- **响应式**：以 375px 移动端为主要设计基准，同时检查 390px 和 430px
- **可访问性**：关键指标文字清晰可辨，表格支持横向滚动

---

## 11. 风险与待确认项

| 风险/问题 | 影响范围 | 处理策略 |
|-----------|----------|----------|
| "秦始皇"品种名称是否准确 | 其他品种 Tab | 已确认：用户确认为正确名称 |
| 组织切换是页面级 Tab 还是表格筛选 | 所有 Tab | 已确认：页面级 Tab |
| 数据表格列数较多，移动端展示效果 | 所有 Tab | 采用横向滚动 + 固定首列 |
| 进度条展示规则（按什么计算） | 所有 Tab | 需后端确认增量贡献的计算口径 |
| 时间筛选默认选中值 | Banner 区域 | 默认当前月/日 |
