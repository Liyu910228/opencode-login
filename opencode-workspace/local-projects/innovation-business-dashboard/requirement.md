# 创新业务看板 需求分析

## 需求类型判断

- **类型**：传统需求
- **演进方式**：0 到 1

## 项目背景

华润雪花创新业务（代工类、特色产品、桶啤）缺少统一的移动端数据查看入口。管理层需要在手机上快速查看各业务分类的销量数据、当期/同期对比和同比增减情况，支持按日/月/年维度切换，按组织层级下钻查看明细排名。

## 目标用户

- 总部管理层
- 事业部管理层
- 营销中心管理层
- 销售大区管理层

## 角色与权限概览

| 角色 | 数据范围 | 可下钻层级 |
|------|----------|------------|
| 总部角色 | 全部事业部 + 全部营销中心 | 事业部 → 营销中心 |
| 事业部角色 | 本事业部下全部营销中心 + 全国各事业部 | 营销中心 → （下一级不可再钻） |
| 营销中心角色 | 本营销中心下全部业务部 + 全国营销中心 | 业务部（末级） |
| 销售大区角色 | 本大区下全部业务部 | 业务部（末级） |

## 核心问题

1. 管理层无法在移动端快速查看创新业务各分类的销量数据
2. 缺少当期/同期/同比增减/同比增幅的直观对比
3. 无法按组织层级下钻查看排名明细

## 目标与非目标

### 目标

- 移动端 H5 看板，支持整体/代工类/特色产品/桶啤四大分类切换
- 支持日/月/年三种时间维度切换（默认月，不支持手动选择具体时间）
- 展示销量（KL）的当期、同期、同比增减、同比增幅
- 支持按事业部/营销中心两个维度切换组织排名
- 支持下钻一级，通过底部弹窗展示更细分的组织数据
- 权限按角色控制数据可见范围

### 非目标

- 不做 PC 端
- 不支持手动选择具体日期/月份/年份
- 特色产品、桶啤的子主题看板结构暂定，本期不实现
- 不做数据导出

## 功能优先级

| 功能 | 优先级 |
|------|--------|
| 整体页签（含整体看板 + 代工类/特色产品/桶啤三个主题看板） | P0 |
| 代工类一级页签（含代工类汇总 + 代加工/包销/定制三个主题看板） | P0 |
| 日/月/年时间维度切换 | P0 |
| 组织维度切换（事业部/营销中心） | P0 |
| 下钻一级（底部弹窗） | P0 |
| 特色产品一级页签 | P1（暂定） |
| 桶啤一级页签 | P1（暂定） |

## 关键流程

1. 用户进入看板 → 默认展示"整体"一级页签 → 默认展示"整体主题看板"
2. 默认时间维度为"月"，可通过 Tab 切换到"日"或"年"
3. 整体页签下通过 Banner 内导航按钮切换：整体看板 / 代工类主题 / 特色产品主题 / 桶啤主题
4. 导航按钮点击后侧边展开选择列表，选中后切换对应主题看板
5. 每个主题看板包含：全国汇总卡片 + 组织排名表格
6. 组织排名表格支持"事业部/营销中心"两个维度 Tab 切换
7. 点击表格行可下钻一级，底部弹窗展示更细分的组织数据

## 非功能要求

- 移动端 H5，适配 375/390/430px 宽度
- 页面加载时间 < 3 秒
- 表格支持横向滚动，不溢出页面

## 风险与待确认项

1. 特色产品、桶啤一级页签的子主题看板结构暂定，需后续确认
2. 同比增幅计算公式：同比增减/同期，最多保留 1 位小数；当同期为 0 时如何展示（待确认）
3. 日维度的同期对比逻辑：当天 vs 去年同日？（待确认）
4. 数据接口来源和刷新频率（待确认）
