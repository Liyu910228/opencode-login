## 2. 页面骨架结论

- **阅读方式**：分屏看板型（顶部 Tab 切换大维度，Tab 内横滑子维度切换具体生意表现）
- **采用理由**：
  - 需求明确为 4 个主 Tab，每个 Tab 内部有多个可切换的子维度标题
  - 子维度包括总览（如"分档次生意表现"）和单品（如"普高及以上生意表现"），不同子维度展示不同内容结构
  - 与营销驾驶舱现有数据看板页面模式一致：顶部 Tab → 维度横滑 → 内容区
- **用户已确认**：分屏看板型 ✅
- **按项目基线推断**：
  - 子维度导航采用横滑标签卡（类似 Banner 胶囊或横滑列表）
  - 表格采用固定首列 + 横向滚动
  - 条形图采用纯 CSS 横向进度条式展示

---

## 3. 页面整体结构

```
顶部导航栏（销量快报 + 返回）
    ↓
页面级 Tab（基础用法：分档次 | 分品牌 | 分主力品种 | 其他品种）
    ↓
Banner 区域（标题 + 时间 + 筛选按钮）
    ↓
子维度横滑导航（各 Tab 内部的具体生意表现切换）
    ↓
内容区（根据子维度动态变化：汇总卡片 / 条形图 / 表格等）
    ↓
组织切换页签（分事业部 | 分营销中心）
    ↓
数据表格（固定首列 + 横向滚动 + 进度条）
```

## 4. 各 Tab 详细结构

### 4.1 分档次 Tab

**子维度横滑列表**（5 项）：
分档次生意表现 | 普高及以上生意表现 | 次高生意表现 | 中档生意表现 | 大主流生意表现

**子维度 1 — "分档次生意表现"（总览）**：
- 档次结构卡片：4 行对比，每行 = 档次名称 + 当期占比 + 同期占比 + 占比差异（颜色区分正负）
- 维度：普高及以上、次高、中档、大主流

**子维度 2-5 — 单品（普高及以上 / 次高 / 中档 / 大主流）**：
- 汇总卡片：当期、同期、同比增幅
- 组织切换页签：分事业部 | 分营销中心
- 数据表格：组织排名 + 当期/同期/同比增减/同比增幅/增量贡献（进度条）

**筛选按钮**：档次筛选（弹窗选择具体档次）
**时间筛选**：年月日

### 4.2 分品牌 Tab

**子维度**：分品牌生意表现（仅 1 项总览）

**内容结构**：
- 汇总卡片（双维度）：中国品牌 | 国际品牌，指标 = 当期、同期、同比增幅
- 分组织分品牌汇总卡片：维度 = 整体 | 中国品牌 | 国际品牌，指标 = 当期、同比增减
- 组织切换页签：分事业部 | 分营销中心
- 中国品牌表格：组织排名 + 当期/同期/同比增减/同比增幅/增量贡献（进度条）
- 国际品牌表格：同上结构

**筛选按钮**：无
**时间筛选**：年月日

### 4.3 分主力品种 Tab

**子维度横滑列表**（7 项）：
分主力品种生意表现 | 喜力生意表现 | 纯生生意表现 | superX生意表现 | 老雪生意表现 | 红爵生意表现 | 勇闯系列生意表现

  - 指标：当期、同期、同比增幅、销量占比（横向进度条展示）
- 主力品种生意表现表格：
  - 维度同上
  - 指标：当期/同期/同比增减/同比增幅/增量贡献（进度条）

**子维度 2-7 — 单品（喜力 / 纯生 / superX / 老雪 / 红爵 / 勇闯系列）**：
- 汇总卡片：当期、同期、同比增幅

- 数据表格：组织排名 + 当期/同期/同比增减/同比增幅/增量贡献（进度条）

**筛选按钮**：品种筛选（弹窗选择具体品种）
**时间筛选**：年月日

## 5. 模块视觉规格

### 5.1 顶部导航栏

- 高度：44px
- 背景：White（#FFFFFF）
- 标题：销量快报，17/Bold，Text-5（#0B0D0F），居中
- 返回按钮：左侧箭头，24x24

### 5.2 页面级 Tab（基础用法）

- 容器：375 x 44，横向排列
- Tab 数量：4 个（分档次 | 分品牌 | 分主力品种 | 其他品种）
- 选中项：16/Medium，Brand-6（#0073E5）
- 未选中项：14/Regular，Text-4（#576675）
- 选中指示线：24 x 2，圆角 1px，Brand-6
- 交互：点击切换 Tab，保留组织页签和时间筛选状态

### 5.3 Banner 区域

- 背景：雪花蓝渐变 `linear-gradient(135deg, #0073E5, #0052A3)`
- 内边距：20px 16px 16px
- 内容：
  - Banner 主标题：当前 Tab 维度名，18/Medium，白色
  - 时间：年月日，12/Regular，白色 80% 透明度，可点击
  - 筛选按钮：描边胶囊样式，12/Regular，白色，仅 分档次/分主力品种/其他品种 显示

### 5.4 子维度横滑导航

- 位置：Banner 下方
- 背景：White（#FFFFFF）
- 高度：40px
- 交互：横向滚动（`overflow-x: auto`），隐藏滚动条
- 单项样式：
  - 选中态：Brand-1（#E6F2FF）背景，Brand-6（#0073E5）文字，14/Medium
  - 未选中态：Fill-1（#F7F7F8）背景，Text-4（#576675）文字，14/Regular
  - 圆角：Radius-pill（999px）
  - 内边距：6px 14px
  - 间距：8px
- 仅分品牌 Tab 无横滑导航（单一维度）

### 5.5 汇总卡片

- 背景：White（#FFFFFF）
- 圆角：Radius-8（8px）
- 外边距：12px 16px
- 内边距：Space-4（16px）
- 布局：横向等分排列
- 指标名称：12/Regular，Text-3（#8A99A8）
- 指标数值：20/Medium，Text-5（#0B0D0F），DIN 字体
- 同比增幅正负色：正 = Success-4（#2EB82E），负 = Danger-4（#F53D3D）
- 分档次总览特殊：指标为"当期占比/同期占比/占比差异"

### 5.6 档次结构卡片（分档次总览专用）

- 背景：White（#FFFFFF）
- 圆角：8px
- 每行：档次名 + 当期占比 + 同期占比 + 占比差异（颜色区分）
- 行高：约 48px
- 分隔线：1px Divider Color（#F0F0F0）

### 5.7 条形图（分主力品种 / 其他品种总览专用）

- 背景：White（#FFFFFF）
- 圆角：8px
- 每行结构：
  - 品种名（左对齐，14/Regular，Text-5）
  - 占比数值（右对齐，14/Medium，Text-5）
  - 横向进度条（高 8px，Fill-2 背景 #F1F1F4，Brand-6 填充）
  - 同比增幅（颜色区分正负）
- 行高：约 56px

### 5.8 组织切换页签（分段器）

- 容器：圆角 4px，1px Brand-6 描边
- 高度：36px
- 外边距：0 16px 12px
- 选中项：Brand-6 背景，白色文字，14/Medium
- 未选中项：透明背景，Brand-6 文字，14/Regular

### 5.9 数据表格

- 背景：White（#FFFFFF）
- 圆角：8px
- 外边距：0 16px 16px
- 表头背景：Brand-1（#E6F2FF）
- 表头文字：11/Regular，Text-4（#576675）
- 数据文本：13/Regular，Text-5（#0B0D0F）
- 固定首列：组织名 + 排名徽章，宽度 88-96px
- 横向滚动：`overflow-x: auto`，隐藏滚动条
- 进度条：高 6px，Fill-2 背景，Brand-6 填充
- 排名 Top3：蓝色圆形徽章（Brand-6 背景，白色数字）
- 正负色：同比增幅/增减按正负着色

## 6. 弹窗交互

### 6.1 筛选弹窗（档次筛选 / 品种筛选）

- 从底部弹出的抽屉（Drawer）
- 遮罩：rgba(0,0,0,0.4)
- 弹窗背景：White，圆角 12px（仅顶部）
- 标题：档次筛选 / 品种筛选
- 选项列表：可单选，选中项带蓝色勾选
- 底部：取消 | 确认

### 6.2 时间选择

- 从底部弹出的日期选择器
- 简化实现：年月日三级选择或预设快捷选项

## 8. 设计冻结规则

- demo 实现阶段不再向用户确认颜色、布局、组件选择或是否继续
- 缺失信息按同类页面、项目画像、Snow Mobile 规范推断
- **明确禁止的脑补**：
  - 不要把业务需求脑补成通用仪表盘
  - 不要使用随机渐变、大面积彩色卡片、emoji 图标
  - 不要只做功能字段完整，却忽略项目真实结构模式
  - 不要用一堆卡片从上到下堆完所有信息，忽略子维度切换
